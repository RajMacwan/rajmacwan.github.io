The lab bench.

Hands-on experiments, tool reviews, home-lab builds, and step-by-step how-tos from my own infrastructure. If I'm writing about it here, I've actually run it.

What this Grid covers

LabGrid is the builder's section. Home-lab architecture, hypervisor and container experiments, network gear write-ups, self-hosted services, tool comparisons, and the how-tos that come out of breaking something and fixing it. The rule for anything posted here is simple: I've run it myself, usually on my own hardware, and the numbers and caveats are honest.

The scope overlaps with InfraGrid and ThreatGrid but the style is different. LabGrid posts are procedural, reproducible, and opinionated about specific products — which router, which firewall distribution, which observability stack, which tool earned its license cost and which didn't. Expect screenshots, config snippets, and post-mortems of the experiments that failed.

If you maintain a home lab for professional growth, or you're evaluating something before putting it in front of production traffic, this is the Grid to watch.
