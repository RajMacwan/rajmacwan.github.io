## Publishing a New Blog Post

1. Copy an existing blog post HTML (e.g., `blog/kev-as-backlog.html`) to a new file
2. Update the title, date, category, and content
3. Add an entry to `blog/index.html` and `blog/feed.xml`
4. Update `sitemap.xml` with the new URL
5. `git add . && git commit -m "post: new blog post" && git push`
6. GitHub Pages rebuilds automatically within ~60 seconds

## Industry Watch

The `industry_watch.py` script fetches RSS from cybersecurity / infrastructure / AI sources and rebuilds `industry-watch.html`. A GitHub Action runs it every 6 hours.

To run manually:
```bash
python3 industry_watch.py
```
